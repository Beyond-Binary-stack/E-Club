## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

1. **Hero** - Eye-catching introduction with CTA buttons
2. **About** - Club introduction and values
3. **What We Do** - Activity cards showcase
4. **Why Join** - Benefits section
5. **Activities Preview** - Weekly, monthly, and annual activities
6. **Community** - Inspirational quote section
7. **Call to Action** - Final encouragement to join
8. **Footer** - Club info and social links
